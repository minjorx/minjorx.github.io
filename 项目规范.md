# 项目规范

## 记账页面开发规范

### 通用样式要求

项目中全局使用相关样式作为通用样式，**开发时只开发结构，不改变样式**。

### 具体要求

1. **全局样式优先**：所有组件样式应优先使用全局定义的 CSS 变量和通用样式类
2. **禁止修改样式**：在开发功能时，只编写 HTML 结构（模板），不编写或修改 `<style>` 样式代码
3. **样式统一管理**：所有样式统一在 `docs/.vuepress/theme/styles/` 目录下管理
4. **遵循设计系统**：使用全局定义的配色、字体、间距等设计变量

### 参考资源

- 全局样式文件：`docs/.vuepress/theme/styles/custom.css`
- 记账组件示例：`docs/.vuepress/theme/components/accounting/Accounting.vue`
