# Web Tools

Welcome to the Web Tools section of Minjor's development resources. This page contains various online tools that can help with web development and design tasks.

## Available Tools

- **Color Palette Generator** - Create beautiful color schemes for your projects
- **CSS Gradient Generator** - Generate smooth gradients for backgrounds
- **Icon Library** - Collection of scalable vector icons
- **Code Formatter** - Format and beautify your HTML, CSS, and JavaScript code
- **Image Optimizer** - Compress images without losing quality
- **Responsive Design Checker** - Test how your site looks on different devices

## How to Use

Each tool is designed to be user-friendly and accessible directly from your browser. Simply navigate to the specific tool page to start using it.
