# 常用命令
## linux
```bash
# 当前目录下所有文件
ls -l
# or
ll

pwd
dir
cat
tail -f

# 切换用户
sudo su - <user>
# - 登录(可exit退出)
```

## docker
```bash
# 镜像相关
# 拉取镜像到本地
docker pull <image_name>

docker run

# 容器相关
# 容器资源监控
docker stats

# 查看容器详情
docker inspect <container_id>

# 查看容器进程
docker top <container_id>

# 查看容器资源使用
docker stats <container_id>
```

## PostgreSQL元命令
```sql
-- 元命令

-- 列出所有数据库
\l

-- 切换数据库 
\c <database>

-- 查看表结构
\d <table_name>

-- show tables
\dt

-- 退出
\q

```
